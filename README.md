# File XOR Processor

Тестовое задание на позицию "Программист C++/Qt".

Приложение предназначено для модификации файлов с использованием операции XOR с 8-байтовым ключом.

## Описание программы
- **Обработка больших файлов**: Поддержка файлов объемом 10 ГБ+ без "замирания" интерфейса благодаря многопоточности (`QThread`).
- **Управление процессом**: Возможность приостановить, возобновить или полностью прервать обработку в любой момент.
- **Безопасность данных**: Обработка файлов "на месте" (в той же папке) реализована через временные файлы, что предотвращает потерю данных при сбоях.
- **Работа по таймеру**: Режим периодического опроса входной папки.
- **Современный UI**: Интерфейс написан на QML (Material Style).

## Технологии
- **Язык**: C++ 17
- **Фреймворк**: Qt 6.x (Quick/QML)
- **Система сборки**: CMake
- **Компилятор**: MinGW 64-bit

## Как собрать
1. Откройте `CMakeLists.txt` в Qt Creator.
2. Выберите комплект сборки с Qt 6.x и MinGW.
3. Выполните "Run CMake" и запустите сборку.

## Тестирование
Для проверки производительности на больших файлах использовалась утилита `fsutil` для создания файлов объемом 10 ГБ. Приложение успешно обрабатывает такие файлы, сохраняя отзывчивость интерфейса.
